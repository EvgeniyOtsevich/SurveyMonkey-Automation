# AutomationTest

### Requirement:
- Java 8
- Maven 3.5.0

### Setup 
1. Clone projects 
```
  > git clone https://github.com/EvgeniyOtsevich/aqa_carbyne.git
```
2. For running tests 
```
  > mvn clean test
```
3. For getting Allure report (serve)
```
  > mvn io.qameta.allure:allure-maven:serve
```
